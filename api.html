Acesse os dados de forma fácil

O nosso portal de transparência possui uma API para facilitar o seu programa a consumir os nossos dados.

Sobre

  • Para consumir os nossos dados basta uma única chamada REST.
  • Não conhece API's RESTFUL ou nunca trabalhou com JSON? Clique aqui e entenda como funciona.

Acesso aos dados

Para acessar os dados basta acessar o endereço:

http://api.instituicao.gov.br/requisicao?tipo=<TIPO>&filtro=<FILTRO>.json

Como destacado no endereço de chamada acima podesse escolher o tipo de dado que se deseja acessar e aplicar um filtro a este dado. Tipos de dados válidos, e seus filtros, são:

  • Convenios: relaciona todos os convênios firmados pela instituição. TIPO: convenios, FILTRO:
    • ativos
    • encerrados
    • todos
  • Projetos: relaciona todos os projetos desenvolvidos pela instituição. TIPO: projetos, FILTRO:
    • ativos
    • encerrados
    • todos
  • Funcionarios: relaciona os funcionários da instituição. TIPO: funcionarios, FILTRO:
    • diretoria
    • gerencia
    • concursados
    • tercerizados
    • estagiarios

Exemplo

A chamada abaixo é um exemplo de chamada que retorna todos os convênios que estão ativos na instituição:

http://api.instituicao.gov.br/requisicao?tipo=convenios&filtro=ativos.json

Retorno obtido

A API do portal de transparência retorna seus dados no formato JSON, sendo que este arquivo pode ser lido facilmente e é facilmente interpretado por programas simples.
